SonarQube Setup for React Application

Local Installation - Step by Step Guide

Step 1: Install Java 17

SonarQube requires Java to run.

Download: Install Java JDK 17

Verify installation:

java -version

You should see Java 17 in the output.

Back to top

Step 2: Download and Start SonarQube

Download: SonarQube Community Edition

  1. Extract the zip file
  2. Navigate to:
sonarqube/bin/windows-x86-64

Start SonarQube:

StartSonar.bat

Open browser:

http://localhost:9000

Login credentials:

  • Username: admin
  • Password: admin

(Change password on first login)

Step 3: Generate SonarQube Token

  1. Go to Profile (top-right corner)
  2. Click My Account ‚Üí Security
  3. Generate a new token
  4. Copy and save the token securely

Step 4: Clone React Application from Git

git clone <your-react-repository-url>
cd your-react-project

Make sure package.json exists in the project root.

Back to top

Step 5: Download and Configure Sonar Scanner

Download: Sonar Scanner

  1. Extract it
  2. Add the bin directory to the system PATH

Verify:

sonar-scanner -v
Back to top

Step 6: Create sonar-project.properties File

Create a file in the project root directory:

sonar-project.properties
sonar.projectKey=react-todo-app
sonar.projectName=React Todo App
sonar.projectVersion=1.0

sonar.sources=src
sonar.language=js

sonar.host.url=http://localhost:9000
sonar.login=YOUR_SONAR_TOKEN

üîπ Important: Replace YOUR_SONAR_TOKEN with your actual token from Step 3.

Step 8: Run Sonar Scanner

Open terminal or VS Code terminal in the project root and run:

If successful, you will see:

EXECUTION SUCCESS

Step 9: View Results in SonarQube UI

Open:

http://localhost:9000

You can now see:

  • Bugs - Potential bugs in your code
  • Vulnerabilities - Security issues
  • Code Smells - Maintainability issues
  • Security Hotspots - Security-sensitive code to review
